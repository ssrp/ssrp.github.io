Baseline Model

For the baseline model, we created a simple artificial neural network architecture which takes 4410 samples (0.1s) from a bad quality mobile phone (specifically Samsung S3) as the input vector and generates one value as the output. This one value corresponds to the 4410th sample of the corresponding condenser mic recorded audio. We are doing this because we believe that the 4410th sample is being dependent on the previous window of 0.1s, which might be helpful to recreate it.

We also aim to create a similar baseline on freq domain.

Proposed Model

For the final model, we are planning to use the Wave-U-Net model to create the high quality recordings.
