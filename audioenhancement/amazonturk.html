User Study for Perceptual Evaluation

After recreating the enhanced audio, we aim to perform a survey using this website (removing all the unwanted information). The user study would contain the same audio file recorded with different microphones and the corresponding enhanced version using the network. The users will be asked to give their preference with some reasons. We expect to have about 15 users for this study.
