## ğŸš€ Getting Started

### Prerequisites

- Node.js (version 18 or higher)
- npm or yarn package manager

### Installation

1. Clone the repository:
```bash
git clone https://github.com/ssrp/ssrp.github.io.git
cd ssrp.github.io
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run develop
```

4. Open your browser and visit `http://localhost:8000`

## ğŸ“ Available Scripts

## ğŸ¨ Customization

### Personal Information
Update your personal information in:
- `gatsby-config.js` - Site metadata
- Component files in `src/components/` - Content and data

### Styling
- Main styles are in `src/scss/`
- Component-specific styles are co-located with components
- Uses Sass for enhanced CSS capabilities

### Content
- Update experience, projects, and skills in their respective component files
- Add new images to `src/img/` directory
- Modify animations and interactions in component JavaScript files
